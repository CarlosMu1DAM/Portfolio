Descripción
Este proyecto consiste en una página web tipo portfolio donde se muestra:
Información personal y profesional
Stack tecnológico
Proyectos desarrollados
Formación académica
Datos de contacto

El objetivo es disponer de una presencia online profesional publicada mediante GitHub Pages.

Características principales:
Diseño moderno con tema oscuro
Cursor personalizado animado
Animaciones de aparición al hacer scroll
Diseño responsive
Estructura modular por secciones
Enlaces externos a GitHub y correo
Uso de variables CSS personalizadas
Animaciones CSS y JavaScript nativo

Tecnologías utilizadas:
HTML5
CSS3 (Custom Properties / Animations / Grid / Flexbox)
JavaScript Vanilla
Google Fonts
GitHub Pages
